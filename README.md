# Password Strength Checker

## Overview
A simple Python project that checks the strength of a password based on length, uppercase, lowercase, numbers, and special characters.

## Features
- Checks password strength
- Gives rating (Very Weak to Very Strong)
- Suggests improvements
- Easy command-line interface

## Requirements
- Python 3.x

## How to Run
```bash
python password_strength_checker.py
```

## Rules Used
- At least 8 characters
- Uppercase letter
- Lowercase letter
- Number
- Special character

## Output Example
```
Enter password: Hello123
Password Strength: Strong
Suggestion: Add special characters
```
